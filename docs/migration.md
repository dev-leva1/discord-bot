# Миграция структуры

## Что изменилось

- Исходники перенесены в `src/`
- Документация в `docs/`
- Конфиги и рабочие файлы в `data/`
- Шрифты в `assets/`

## Проверка после миграции

1. Установите зависимости:

```
uv pip install -e .[dev]
```

2. Проверьте запуск:

```
python bot.py
```

3. Запустите тесты:

```
uv run pytest
```
