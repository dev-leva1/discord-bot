# Обзор

Этот репозиторий содержит многофункционального Discord-бота с модульной архитектурой. Код находится в `src/`, конфигурации и рабочие данные — в `data/`.

## Основные модули

- `src/app` — точка входа и контейнер зависимостей.
- `src/cogs` — команды и события.
- `src/application` — контракты и бизнес-логика.
- `src/infrastructure` — хранилища и интеграции.
